# Rough Idea

I want a way to manage my Claude Code sessions via a web app. I've seen cute themes like a little office building of agents where there are little people doing tasks that represent the things that the agents are doing. The main thing would be giving the agent a task and having it be on its own (unless it wants to ask a question) — this would help me manage my agents to work on multiple projects at the same time. Like https://cubicle.run/ but I want to do it myself for the flexibility.
